# Cursor Prompt – Event Explorer

## Task

Implement or improve astronomy event discovery.

## Features

- Event listing with date range and filters
- Meteor showers, eclipses, conjunctions, comet events
- Event detail pages with description and dates
- Save/reminder-ready interaction (extension point)
- Homepage integration for upcoming events

## Reference

Routes: `/events`, `/events/:id`. Data: `src/lib/discovery/eventAwareness.ts` (EVENTS_2026). Types: `src/lib/discovery/types.ts` (CelestialEvent).
