# Alnitar Product Requirements Document

## Vision

Alnitar is a premium astronomy discovery ecosystem that enables anyone to explore, understand, and observe the night sky through modern interfaces, curated knowledge, and intelligent discovery tools.

## Core Product Areas

1. **Sky Discovery** — Scan, identify, and explore the night sky.
2. **Celestial Knowledge** — Object encyclopedia, taxonomy, educational content.
3. **Event Exploration** — Meteor showers, eclipses, conjunctions, visibility events.
4. **Observation Journaling** — Personal log, notes, timestamps, optional media.
5. **Community and Sharing** — Public profiles, astrophotography, shareable observations.

## Primary User Journeys

### Journey 1 – Discover the Sky

User opens Alnitar and quickly explores what objects are visible tonight.

### Journey 2 – Learn About Objects

User searches or browses celestial objects and reads structured scientific explanations.

### Journey 3 – Track Astronomy Events

User discovers upcoming events such as meteor showers or eclipses.

### Journey 4 – Record Observations

User logs observations and builds a personal astronomy journal.

## MVP Scope

- Premium homepage
- Celestial object explorer
- Astronomy event explorer
- Object detail pages
- Event detail pages
- Favorites and save system
- Observation journal architecture
